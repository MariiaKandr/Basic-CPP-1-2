# 📘 Базовый курс C++

Добро пожаловать в репозиторий с материалами по курсу! Здесь собраны конспекты, примеры кода и практические задания для тех, кто начинает изучение языка C++ с нуля или хочет систематизировать свои знания.

---

## 🎯 Цель курса

Познакомить слушателей с фундаментальными возможностями C++: от переменных и типов данных до объектно-ориентированного программирования и работы с динамической памятью. После освоения материалов вы сможете:

- писать небольшие консольные приложения;
- понимать разницу между стеком и кучей;
- использовать указатели и ссылки;
- создавать классы и работать с наследованием/полиморфизмом.

## 🚀 Как пользоваться репозиторием

### 1. Онлайн-просмотр
Просто откройте любой `.md` файл на GitHub — содержимое отобразится с поддержкой Markdown.

### 2. Клонирование репозитория
```bash
git clone https://github.com/MariiaKandr/Basic-CPP-1-2.git
cd Basic-CPP-1-2
```

### 3. Рекомендуемый порядок изучения
Изучайте модули **по порядку** — каждый следующий опирается на предыдущие. Для каждого модуля:
1. Прочитайте `.md`-файл с теорией.
2. Поэкспериментируйте: измените код, добавьте свои примеры.
3. Попробуйте решить предложенные задачи из [`Задачник.md`](https://github.com/MariiaKandr/Basic-CPP-1-2/blob/main/%D0%97%D0%B0%D0%B4%D0%B0%D1%87%D0%BD%D0%B8%D0%BA.md).

### 4. Как проверять себя
- Попробуйте **написать код с нуля** по описанию из `.md`, не подглядывая в готовые примеры.
- Измените существующие примеры: добавьте новый функционал, исправьте ошибку (намеренную — потом проверьте).
- Используйте отладчик (GDB, встроенный в IDE), чтобы понять, как работают указатели и динамическая память.

## 🛠️ Требования к окружению

- Компилятор с поддержкой **C++11** (рекомендуется C++17).
- Любой текстовый редактор или IDE (Visual Studio, CLion, VS Code, Qt Creator, Code::Blocks).
- Базовое умение работать с командной строкой (для самостоятельной компиляции).

Если вы абсолютный новичок, начните с онлайн-компилятора — это проще.
